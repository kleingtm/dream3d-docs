Region Growing (ImageProcessing) {#itkregiongrowing}
=====

## Group (Subgroup) ##
Unsupported (Misc)


## Description ##

| Name             | Type |

## Created Arrays ##

|------|--------------------|-------------|---------|
| uint8_t | ProcessedArray | 8 bit image data | |
